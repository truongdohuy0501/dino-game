Put your character PNG files here (transparent background).

Required for idle + happy:
  dino-idle-1.png
  dino-idle-2.png
  dino-happy.png

Optional for feeding animation:
  dino-eat-1.png
  dino-eat-2.png
  dino-eat-3.png
  dino-eat-4.png

DALL-E images are often 1024x1024 — that is OK.
The game scales them to 360x280 on screen automatically.

After adding files:
  npm run dev
Then hard refresh browser: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
